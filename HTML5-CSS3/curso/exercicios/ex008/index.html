Principais formatações

Negrito/destaque

Nesta frase temos um termo em negrito usando uma tag "B" (não semântica!).

Nesta frase, tenho um termo em destaque usando a tag STRONG (semântica!). ⭐

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag (não semântica).

Nesta frase, temos um termo em ênfase usando a tag EM (semântica). ⭐

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

E no outro parágrafo, temos um texto marcado no final.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo utilizando as tags BIG(obsoleta) e SMALL.

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado, com a tag DEL (semantica).

Texto inserido

Podemos parcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois com a tag INS (semântica).

Existe também um texto sublinhado com a tag U (não semântica).

Texto sobrescrito

Para inserir coisas do tipo x20+3 utilizando a tag SUP

Texto subscrito

Para inserir coisas do tipo H2O Utilizando a tag SUB
